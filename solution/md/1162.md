#### [1162. 地图分析](https://leetcode-cn.com/problems/as-far-from-land-as-possible/)

#### 算法：

要求返回距离陆地区域最远的海洋区域到陆地的最短距离。

也就是求陆地到海洋的最短距离，然后在不同海洋区域中取距离的最大值。因为边权全为 1，所以可以使用 BFS 求最短路径（多源 BFS）。

#### 时间复杂度分析：

O(m * n)

#### 代码：

```java
class Solution {
    private int[][] direction = {{1, 0}, {-1, 0}, {0, 1}, {0, -1}};
    public int maxDistance(int[][] grid) {
        int rows = grid.length, cols = grid[0].length;
        int[][] dist = new int[rows][cols];
        boolean[][] st = new boolean[rows][cols];
        Queue<int[]> queue = new ArrayDeque<>();
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                if (grid[i][j] == 1) {
                    queue.offer(new int[] {i, j});
                    st[i][j] = true;
                }
            }
        }
        // 如果只有陆地或者海洋，返回 -1
        if (queue.size() == 0 || queue.size() == rows * cols) return -1;
        
        while (!queue.isEmpty()) {
            int[] pair = queue.poll();
            int i = pair[0], j = pair[1];
            for (int[] dir : direction) {
                int x = i + dir[0], y = j + dir[1];
                if (x < 0 || x >= rows || y < 0 || y >= cols
                        || st[x][y]) continue;
                dist[x][y] = dist[i][j] + 1;
                queue.offer(new int[] {x, y});
                st[x][y] = true;
            }
        }

        int ret = 0;
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                if (grid[i][j] == 0) ret = Math.max(ret, dist[i][j]);
            }
        }

        return ret;
    }
}
```
